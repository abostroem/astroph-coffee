HELP: basic search usage
Search type How it's done Examples
Full-text keyword search Just type in the keywords you want to use. To look for alternate forms of keywords, use the * (asterisk) operator. All keywords separated by spaces will be required to appear in the results. They will be matched against content fields in the database if no content fields are provided (see below). gaussian process regression
Kepler exoplanet*
(matches exoplanet, exoplanets, exoplanetary)
Boolean operators Use NOT keyword to make sure keyword doesn't show up in the results. Use keyword1 OR keyword2 to match either keyword1 or keyword2. Use keyword1 AND keyword2 to make sure both or all keywords show up in the results. You can use parentheses to specify the operator precedence. Kepler NOT transit*
(K2 OR Gaia) AND supernova*
HELP: advanced search using content fields
Search type How it's done Examples
Author content field Use authors:author_name. authors:Lupton authors:Gunn NOT SDSS
authors:Bakos NOT planet*
Title content field Use title:keyword to search for keyword in title. All of the rules above for Booleans and multiple matching work here too. title:Gaia authors:Eyer transients
title:LSST authors:Juric
Abstract content field Use abstract:keyword to search for keyword in the article abstract. All of the rules above for Booleans and multiple matching work here too. The * operator will be particularly useful here. abstract:transient* title:PTF NOT title:supernova*
arXiv ID content field Type in an arXiv ID (without the 'arXiv:' prefix) to look up the paper: arxiv_id:XXXX.YYYY arxiv_id:1601.05121
Article type content field Use either article_type:astronomy or article_type:crosslists to restrict results to the respective listings. abstract:plasma title:sun article_type:astronomy
